Estudo Autónomo: Início do Portfólio

Objetivo: Configurar o ambiente de desenvolvimento e criar a estrutura inicial do teu projeto pessoal.

Tempo estimado: 2h - 3h

1 Preparar as Pastas

A organização é fundamental. Vamos criar a "casa" do teu projeto.

  1. Cria uma pasta no teu computador chamada CursoFrontend.
  2. Dentro dessa pasta, cria uma nova pasta chamada Portfolio.
  3. Abre o VS Code.
  4. Vai a File > Open Folder e seleciona a pasta Portfolio.

2 O Ficheiro Index

Todo o site começa com um ficheiro index. Cria o teu primeiro ficheiro HTML.

  1. No VS Code, cria um novo ficheiro chamado index.html (tudo em minúsculas!).
  2. No ficheiro, escreve apenas ! e pressiona Enter ou Tab.
  3. O VS Code vai gerar automaticamente a estrutura base do HTML5.
<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>Document</title> </head> <body> </body> </html>

3 Personalizar o Conteúdo

Agora, vamos adicionar o teu conteúdo inicial.

  1. Muda o texto dentro de <title> para Portfólio de [Teu Nome].
  2. Muda o atributo lang="en" para lang="pt".
  3. Dentro do <body>, adiciona o seguinte:
    • Um título <h1> com o teu nome.
    • Um parágrafo <p> com uma breve descrição sobre ti (ex: "Olá! Sou um estudante de Frontend...").
    • Um subtítulo <h2> a dizer "Meus Objetivos".
    • Uma lista <ul> com 3 coisas que queres aprender neste curso.

4 Testar com Live Server

Não abras o ficheiro clicando duas vezes! Vamos usar o servidor.

  1. Clica com o botão direito no ficheiro index.html dentro do VS Code.
  2. Seleciona a opção "Open with Live Server".
  3. O teu browser deve abrir automaticamente com a tua página.
  4. Teste Mágico: Mantém o browser e o VS Code lado a lado. Muda o texto do <h1> e grava (Ctrl+S). Vê se o browser atualiza sozinho!
Dica Pro: Se não vires a opção "Open with Live Server", verifica se instalaste a extensão na aba de extensões (ícone dos quadrados na esquerda).
